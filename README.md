Mini Fintech Dashboard

A full-stack MERN application for tracking income and expenses.

Features

-Add transactions
-Delete transactions
-Track balance
-View total income
-View total expenses
-Income vs Expense chart
-MongoDB Atlas database

Tech Stack

Frontend
  -React.js
  -Axios
  -Chart.js
  
Backend
 -Node.js
 -Express.js
 
Database
  -MongoDB Atlas
  -Mongoose
  
Installation

Backend
  cd backend
  npm install
  npm run dev

  
Frontend
  cd frontend
  npm install
  npm start
